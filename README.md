# 🍔 Food Delivery Lanches

Projeto de uma landing page para um serviço de delivery de lanches, desenvolvido com foco em praticar habilidades de desenvolvimento web front-end, design moderno e responsividade.

---

## 🚀 Acesse o projeto

Se estiver hospedado no Vercel ou outro serviço:

👉https://food-delivery-lanches.vercel.app/

---

## 🧠 Objetivo

O objetivo deste projeto é simular um site real de delivery, aplicando conhecimentos de desenvolvimento web, com foco em:

- Estruturação de páginas com HTML5
- Estilização moderna com CSS3
- Interatividade com JavaScript (se houver)
- Design responsivo para mobile e desktop
- Organização de componentes e layout

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (opcional)
- Flexbox
- Grid Layout

---

## ✨ Funcionalidades

- Página inicial com apresentação do serviço
- Seção de cardápio com lanches e bebidas
- Layout responsivo para diferentes telas
- Interface moderna e agradável
- Estrutura organizada e fácil de manter

---

## 📌 Melhorias futuras

- Sistema de pedidos online
- Integração com WhatsApp para pedidos
- Carrinho de compras funcional
- Sistema de login e cadastro de usuários
- Backend para gerenciamento de pedidos
- Painel administrativo
